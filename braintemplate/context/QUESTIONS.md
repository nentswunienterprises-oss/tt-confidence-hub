# Context Questions

The current state and priorities of the system at this moment.

## Active Context

- **What is the current primary goal or focus area?**
- **Why does this goal matter right now?**
- **What modules, files, or systems are relevant to this goal?**
- **Why is each relevant module important for this goal?**
- **What assumptions are we taking as given or true?**
  - What do we assume about the user's intent?
  - What do we assume about the technical environment?
  - What do we assume about existing functionality?
- **What is explicitly NOT a goal or not in scope right now?**
- **What are the highest-priority tasks or milestones?**
  - What is blocking other work?
  - What delivers the most value first?
  - What is the correct sequence?
- **Who is driving this work, and when was the context last updated?**
- **Are there open questions or unknowns that need resolution?**
- **What information is missing that would help clarify the path?**
