CLARITY PHASE Drill
(Fully Controlled Session System)
Objective (non-negotiable)

Student must SEE the problem clearly before trying to solve it

This means:

can name terms
can identify type
can state steps
can explain why
Session Structure (FIXED)

Clarity is NOT freestyle teaching.

It is:

Controlled Modeling → Immediate Recognition → Guided Repetition

SESSION = 3 SETS

Always.

No variation.

SET 1 -MODELING SET
Purpose: Build mental map
Structure
2 problems
tutor models BOTH using 3-Layer Lens
Rules
slow
explicit
no skipping language
student must TRACK (not just watch)
What tutor does

For EACH problem:

Vocabulary
Method
Reason

Then:

“Explain it back to me.”

Tutor selects:

Vocabulary recognition

☐ clear steps

☐ vague
☐ clear logic

☐ actively tracking
Output of Set 1

Clarity Exposure Score

SET 2 -IDENTIFICATION SET
Purpose: Can they SEE without solving?

This is where most tutors mess up.

We DO NOT solve yet.

Structure
4 problems
student does NOT solve fully

They must:

name terms
identify type
state steps
What tutor says

For each problem:

“What is this?”
“What type?”
“What are the steps?”
“Why does this method work?”

☐ clear

☐ confident
Output of Set 2

Clarity Recognition Score

SCORING SYSTEM (CLARITY)

We keep this clean and ruthless.

No overthinking.

Each Set Produces a Score (0–2)
Scoring Scale
0 = weak
1 = partial
2 = clear
SET 1 -MODEL (Exposure Score)

Based on:

vocabulary
method
reason
engagement
Rule:

If majority = weak → 0
If mixed → 1
If mostly clear → 2

SET 2 -IDENTIFICATION (Recognition Score)

Based on:

type recognition
step recall
reason recall
response behavior
Rule:

Same:

weak → 0
partial → 1
clear → 2
SET 3 -APPLY (Application Score)

Based on:

start behavior
step execution
vocabulary usage
reason usage
Rule:
weak → 0
partial → 1
clear → 2
CORE SESSION SCORE

We weight them:

Final Score =

(Set 1 × 1)
+ (Set 2 × 2)
+ (Set 3 × 2)

MAX = 10
Why this weighting
Seeing matters (Set 1)
Recognizing matters more
Applying matters most
